# Final Investigation Report

## Verdict

No ransomware was executed.

The investigation demonstrated how a Security Operations Center (SOC) analyst can identify ransomware-related behaviors using Windows telemetry.

## Detection Content

- Sigma Rule
- KQL Query
- IOC Documentation
- MITRE ATT&CK Mapping

## Lessons Learned

- Early detection reduces impact.
- Sysmon provides valuable telemetry.
- LOLBins are frequently observed during ransomware incidents.
- PowerShell monitoring is essential.
