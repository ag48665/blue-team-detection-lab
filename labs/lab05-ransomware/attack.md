# Attack Simulation

## Objective

Simulate the investigation of a ransomware incident without executing real malware.

---

## Scenario

The analyst receives an alert indicating possible ransomware activity on a Windows workstation.

Observed behaviors include:

- PowerShell execution
- LOLBin execution
- File modification activity
- Suspicious command-line arguments

---

## ATT&CK Tactics

- Initial Access
- Execution
- Defense Evasion
- Impact

---

## Goal

Identify attacker behavior and create detection opportunities.
