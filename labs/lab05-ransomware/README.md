# Lab05 – Ransomware Investigation

## Objective

Investigate a simulated ransomware incident using Windows logs and detection engineering techniques.

---

## Scenario

A Windows endpoint generated multiple suspicious events that may indicate ransomware activity.

The objective is to identify suspicious behavior, document Indicators of Compromise (IOCs), map the activity to MITRE ATT&CK, and propose detection opportunities.

---

## Skills Demonstrated

- Incident Investigation
- IOC Analysis
- MITRE ATT&CK Mapping
- Detection Engineering
- Windows Event Analysis
- Ransomware Detection

---

## Tools

- Sysmon
- Event Viewer
- Sigma
- Microsoft Sentinel (KQL)

---

## Deliverables

- Incident Report
- Detection Logic
- IOC Documentation
- MITRE ATT&CK Mapping
- Security Recommendations
