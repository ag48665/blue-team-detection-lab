## 📖 About

This repository contains hands-on Blue Team labs focused on:

- Security Operations Center (SOC)
- Threat Hunting
- Detection Engineering
- Incident Response
- Malware Analysis
- Windows Forensics
- MITRE ATT&CK Mapping
- Sigma Rules
- YARA Rules
- KQL Queries

Each lab contains:

- Objective
- Attack simulation
- Detection
- Investigation
- MITRE ATT&CK Mapping
- Indicators of Compromise (IOCs)
- Screenshots
- Detection Rules
- Lessons Learned
